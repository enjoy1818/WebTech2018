Linked-List

ลิงค์ลิสต์

ลิงค์ลิสต์ เป็นโครงสร้างข้อมูลที่ประกอบด้วยกลุ่มโหนด(Nodes)ที่มีการเรียงลำดับ
โดยที่เราสามารถนำลิงค์ลิสต์มาใช้ในการสร้าง Linear หรือ Non-linear structures ก็ได้
ลิงค์ลิสต์ แบ่งได้เป็น 2 ประเภท คือ
1. Singly Linked List : ลิงค์ลิสต์ที่ชี้ไปในทิศทางเดียว
2. Doubly Linked List : ลิงค์ลิสต์ที่ชี้ในสองทิศทาง (ไป-กลับ)

โหนด(Nodes)

โหนด เป็นโครงสร้างที่ประกอบด้วย 2 ส่วน คือ
1. ข้อมูล(Data) : ข้อมูลในโหนดอาจประกอบด้วยข้อมูลชุดเดียว ข้อมูลหลายชุด หรือข้อมูลหนึ่งชุดที่ประกอบด้วยข้อมูลย่อยหลายชุดก็ได้
2. ลิงค์(Link) : ลิงค์ทำหน้าที่เชื่อมโหนดนั้นกับโหนดอื่นเข้าด้วยกัน เราสามารถใช้ pointer มาช่วยในการสร้างลิงค์ได้

ลิงค์ลิสต์ทุกอันจะต้องมีโหนดที่เรียกว่า "Head Node" โดยที่ Head Node จะเป็นตัวเก็บข้อมูลรายละเอียดเกี่ยวกับลิสต์เช่น จำนวนสมาชิกของลิสต์ ลิงค์ที่ชี้ไปยังโหนดแรก

Create List

ขั้นแรก เราต้องทำการจองพื้นที่สำหรับ Head Node ก่อน
ขั้นที่สอง เราต้องกำหนด head ของ Head Node ให้เป็น null
ขั้นที่สาม เราต้องกำหนดค่า count ของ Head Node ให้เป็น 0

Singly Linked List

Create Data Node

ขั้นแรก สร้าง pointer ที่ชี้ไปยังหน่วยความจำที่เราจองไว้สำหรับโหนด
ขั้นที่สอง กำหนดค่าของข้อมูลในโหนด
ขั้นที่สาม กำหนด next ของโหนดให้เป็น null
ขั้นที่สี่ คืนค่าของ pointer นั้นออกมา

Insertion : Inserting Nodes at the Front of a List

ขั้นแรก สร้างโหนดใหม่ขึ้นมาแล้วให้ pointer ที่ชื่อว่า pNew ชี้ที่โหนด
ขั้นที่สอง กำหนด next ของโหนดใหม่ให้เป็นตำแหน่งที่ head ของ Head Node ชี้อยู่
ขั้นที่สาม กำหนด head ของ Head Node ให้ไปชี้ที่โหนดใหม่
ขั้นที่สี่ เพิ่มค่า count ของ Head Node

Insertion : Inserting Nodes at the End of a List

ขั้นแรก สร้างโหนดใหม่ขึ้นมาแล้วให้ pointer ที่ชื่อว่า pNew ชี้ที่โหนด
ขั้นที่สอง เราต้องใช้ pointer ท่องเข้าไปใน list จนกว่าจะเจอตัวสุดท้าย (ตัวที่ next เป็น null) สมมติว่า pointer นี้ชื่อ start
ขั้นที่สาม กำหนดให้ next ของโหนดที่ start ชี้อยู่ไปชี้ที่ pNew
ขั้นที่สี่ เพิ่มค่า count ของ Head Node

Deletion

การลบโหนดใน Linked List นั้นมีอยู่ 3 กรณี
1. กรณีที่ต้องการลบโหนดที่อยู่หัวลิสต์
ให้ pointer start ชี้ไปที่ root.head -> ให้ root.head ชี้ไปที่ start.next -> ลบ start ทิ้ง
2. กรณีที่ต้องการลบโหนดที่อยู่ระหว่างลิสต์
ให้ pointer prev ชี้ไปที่ root.head และ pointer start ชี้ไปที่ prev.next -> วนลูปให้ prev กับ start วิ่งไปด้วยกันไปจน start เจอโหนดที่ต้องการลบ
-> ให้ prev.next ชี้ไปที่ start.next -> ลบ start ทิ้ง
3. กรณีที่ต้องการลบโหนดที่อยู่ท้ายลิสต์
ห้ pointer prev ชี้ไปที่ root.head และ pointer start ชี้ไปที่ prev.next -> วนลูปให้ prev กับ start วิ่งไปด้วยกันไปจน start เจอโหนดสุดท้าย
-> กำหนดให้ prev.next เป็น null -> ลบ start ทิ้ง

Doubly Linked List

Doubly Linked List คือลิงค์ลิสต์ที่มีลิงค์ 2 ทิศทาง คือ จากหน้าไปหลัง และ หลังไปหน้า

Inserting Nodes into a Doubly Linked List



ต้องการแทรกโหนด pNew ไว้หน้า start
ขั้นแรก ให้ pNew.next ชี้ไปที่ start
ขั้นที่สอง ให้ pNew.back ชี้ไปที่ start.back
ขั้นที่สาม ให้ start.back.next ชี้ไปที่ pNew
ขั้นที่สี่ ให้ start.back ชี้ไปที่ pNew

Deleting Nodes from a Doubly Linked List



ต้องการลบโหนด start
ขั้นแรก ให้ start.back.next ชี้ไปที่ start.next
ขั้นที่สอง ให้ start.next.back ชี้ไปที่ start.back
