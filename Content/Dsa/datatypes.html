Data Types

ชนิดข้อมูล

ชนิดข้อมูลในโปรแกรมคอมพิวเตอร์

ชนิดข้อมูลในโปรแกรมคอมพิวเตอร์สามารถจำแนกได้เป็น 2 ประเภท คือ
1. ข้อมูลเชิงเดี่ยว (Atomic Data Type)
ข้อมูลเชิงเดี่ยวคือ ข้อมูลที่ประกอบด้วยค่าเพียงค่าเดียว เช่น int, char, boolean
2. ข้อมูลเชิงประกอบ (Composite Data Type)
ข้อมูลเชิงประกอบคือ ข้อมูลที่ประกอบด้วยข้อมูลเชิงเดี่ยวหลายๆตัวสามารถแตกออกมาเป็นฟิลด์ย่อยได้ เช่น string, array

ชนิดข้อมูล

ชนิดข้อมูลนั้นประกอบด้วย 2 ส่วน คือ
1. กลุ่มค่าของข้อมูล (A set of values)
2. การดำเนินการกับข้อมูล (A set of opertations on values)

ตัวอย่างเช่น ข้อมูลชนิดเลขจำนวนเต็ม (Integer) สามารถแบ่งได้เป็น
ค่าของข้อมูล : จำนวนเต็มบวกที่อยู่ในช่วงที่กำหนด
การดำเนินการกับข้อมูล : บวก, ลบ, คูณ, หาร
