Binary Search Tree

Binary Search Tree เป็น Binary Tree ประเภทหนึ่งโดยที่
ทุกโหนดใน left subtree ต้องมีค่าน้อยกว่า root
ทุกโหนดใน right subtree ต้องมีค่ามากกว่าหรือเท่ากับ root
ในแต่ละ subtree ต้องคงคุณสมบัติของ binary search tree

Search

Find the Smallest Node

การหาค่าที่น้อยที่สุดใน BST สามารถทำได้ด้วยการท่องไปในโหนดที่อยู่ทางซ้ายไปเรื่อยๆจนถึงโหนดสุดท้าย

Find the Largest Node

การหาค่าที่มากที่สุดใน BST สามารถทำได้ด้วยการท่องไปในโหนดที่อยู่ทางขวาไปเรื่อยๆจนถึงโหนดสุดท้าย

Searching a Given Node in a BST

การหาโหนดที่ต้องการใน BST สามารถทำได้ด้วยการท่องเข้าไปในโหนดจนถึงตัวสุดท้ายอย่างมีเงื่อนไข คือ
ถ้าตัวที่ต้องการหามีค่าน้อยกว่าโหนดปัจจุบันให้ไปดูโหนดด้านซ้าย
ถ้าตัวที่ต้องการหามีค่ามากกว่าโหนดปัจจุบันให้ไปดูโหนดด้านขวา

Insertion

เราสามารถแทรกข้อมูลเข้าไปใน BST โดยตามกิ่งไปเรื่อยๆจนเจอ subtree ที่ว่างแล้วแทรกข้อมูลใหม่เข้าไป (แทรกจาก leaf node)




Deletion

การลบข้อมูลจาก BST ต้องทำการค้นหาโหนดที่ต้องการจะลบให้ได้ก่อน ซึ่งโหนดเหล่านั้น แบ่งลักษณะได้เป็น 4 กรณี คือ
1. โหนดที่จะลบเป็น leaf node (ไม่มีลูก)
ลบได้เลย
2. โหนดที่จะลบ มีเพียง Right subtree
เปลี่ยนให้พ่อของโหนดนั้นชี้ไปที่ Right subtree แทน แล้วลบโหนดนั้น
3. โหนดที่จะลบ มีเพียง Left subtree
เปลี่ยนให้พ่อของโหนดนั้นชี้ไปที่ Left subtree แทน แล้วลบโหนดนั้น
4. โหนดที่จะลบมีทั้ง Right และ Left subtree
คัดลอกข้อมูลมากที่สุดของ Left subtree มาที่โหนดที่จะลบ แล้วลบโหนดมากสุดของ Left subtree แทน
