Array & Linked List

อาเรย์และลิ้งค์ลิสต์

ก่อนที่จะเข้าสู่เรื่องอาเรย์และลิ้งค์ลิสต์เราต้องรู้จักกับหน่วยความจำในความพิวเตอร์ก่อน
หน่วยความจำในคอมพิวเตอร์นั้นสามารถเทียบได้กับตู้ล็อกเกอร์ที่มีช่องเก็บของหลายๆช่อง
ก่อนที่เราจะใช้ช่องเก็บของนั้นได้ เราต้องทำการจอง(Allocate)ช่องนั้นๆไว้ก่อน
โดยในคอมพิวเตอร์สามารถแบ่งการจองพื้นที่ได้เป็น 2 แบบ คือ
1. การจองพื้นที่แบบสแตติค (Static Memory Allocation)
2. การจองพื้นที่แบบไดนามิค (Dynamic Memory Allocation)

้

การจองพื้นที่แบบสแตติค (Static Memory Allocation)

การจองพื้นที่แบบสแตติค เกิดขึ้นในช่วงของการ Compile
โดยคอมพิวเตอร์จะจัดสรรหน่วยความจำให้กับตัวแปรแต่ละตัวล่วงหน้า ในขนาดที่คงที่
เมื่อมีการประมวลผล เนื้อที่เหล่านี้ไม่สามารถขยายหรือลดลงได้ ไม่ว่าเราจะได้ใช้หรือไม่ก็ตาม
การจองพื้นที่แบบสแตติคจึงเป็นการใช้พื้นที่หน่วยความจำอย่างไม่มีประสิทธิภาพ สิ้นเปลืองโดยไม่จำเป็น
ตัวอย่างการจัดเก็บข้อมูลที่ใช้การจองพื้นที่แบบสแตติกคือ Array

้

การจองพื้นที่แบบไดนามิค (Dynamic Memory Allocation)

การจองพื้นที่แบบไดนามิค คือการจองพื้นที่โดยที่เราสามารถกำหนดตัวแปรไว้ก่อน
แต่ยังไม่ต้องจัดสรรเนื้อที่หน่วยความจำจนกว่าจะถึงช่วง run time (ตอนประมวลผลโปรแกรม)
มีความยืดหยุ่นและจัดการหน่วยความจำได้อย่างมีประสิทธิภาพมากกว่าการจองพื้นที่แบบสแตติค
ตัวอย่างการจัดเก็บข้อมูลที่ใช้การจองพื้นที่แบบไดนามิคคือ Linked List
