Traverse graph




ในเมื่อข้อมูลเป็นกราฟที่มี Node เชื่อมกันแล้วก็ต้องมาการหาเส้นทางจาก Node หนึ่งๆไปยัง Node อื่นๆ โดย การหาเส้นทางเรียกว่า การท่องกราฟ (Graph Traversal) และการท่องกราฟนั้นก็ยังมีหลายวิธีเช่น BFS, DFS

Breadth-first Traversal

ท่องเข้าไปยังโหนดข้างเคียงทั้งหมดก่อน ค่อยท่องเข้า ไปยังโหนดในระดับต่อไป นำ Queue มาช่วยในการท่องเข้าไปในกราฟ


Depth-first Traversal

ท่องเข้าไปในโหนดลูกหลานของโหนดแรกก่อน ค่อย ย้ายไปท่องโหนดข้างเคียง นำ Stack มาช่วยในการท่องเข้าไปในกราฟ


ความแตกต่างของ BFS กับ DFS ก็แยกกันไม่ยาก
