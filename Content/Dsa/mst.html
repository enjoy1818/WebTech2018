Minimum Spanning Tree ( MST)


เป็นโครงสร้างต้นไม้ที่ประกอบด้วย Node ทั้งหมดในกราฟ รับประกันว่าค่าน้ าหนักรวมของ Edge น้อยที่สุด สามารถสร้าง Minimum Spanning Tree จากกราฟได้ดังนี้ กำหนดเวอร์เทกซ์เริ่มต้น หา Node ข้างเคียงของ Node ที่มี แล้วเลือกใช้เส้นทางที่มีค่า น้ำหนักน้อยที่สุด ซึ่งต้องไม่ใช่เส้นทางที่เชื่อมไปยัง Node ที่เคยเลือก มาแล้ว ทำไปเรื่อยๆ จนครบทุก Node ที่มีในกราฟ
