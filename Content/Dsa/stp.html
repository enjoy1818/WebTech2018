คือการเลือกท่องกราฟโดยพอเมื่อเดินครบทุกโหนดแลล้วจะได้ผลรวมของเส้นทางน้อยที่สุด มีหลายอัลกอลิทึมที่จะไหา shortest path โดยเราจะยกตัวอย่างเป็น Dijktra algorithm Dijktra algorithm คืออัลกอลิทึมที่สามารถค้นหาเส้นทางสั้นที่สุดระหว่างเวอร์เทกซ์ใดๆ ซึ่งมีหลักการดังนี้ แทรก Node แรก เข้าไปในต้นไม้ ให้พิจารณาทุก Node ในต้นไม้ แล้วให้เลือก Edge ที่มีค่าผลรวม Path น้อยที่สุด ทำไปเรื่อยๆ จนกว่าจะครบทุก Node
